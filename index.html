Estruturas de Dados

Estruturas de dados contruídas em C. Foi um projeto pessoal com o objetivo de revisar conceitos aprendidos no início do curso de Ciência da Computação. Contém implementações de pilha, fila, lista, árvore binária simples, árvore rubro-negra e avl.

Github API Report

Trabalho final da disciplina de Banco de Dados II. Esse projeto inclui o consumo da API do Github, bem como a geração de um relatório AD-HOC com base nas informações armazenadas no banco. Nesse projeto foi utilizado o SGDB Postgres SQL.
